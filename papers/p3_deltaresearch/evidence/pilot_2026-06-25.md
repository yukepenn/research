# P3 DeltaResearch — pilot evidence (2026-06-25)

Real models via subscription CLIs (no API cost): `claude`, `codex` (gpt-5.5). 18 controlled worlds
(3 seeds × 6 effectful delta types), programmatic gold A/U/N/C (independent world-physics).
72 LLM episodes in the immutable ledger (paper_id=p3_deltaresearch). Deterministic baselines
computed once per world.

## Result — Affected-Claim Recall (ACR) and Unaffected-Claim Preservation (UCP)
| arm | ACR | UCP | note |
|---|---|---|---|
| oracle (upper bound) | 1.00 | 1.00 | gold |
| **typed_graph (our method, deterministic)** | **1.00** | **1.00** | dominates |
| full_regen (deterministic) | 1.00 | **0.00** | catches all, destroys every preserved claim |
| naive_ledger / flat (deterministic) | **0.13** | 0.97 | misses ~87% of downstream |
| Claude naive LLM | 0.39 | 1.00 | misses ~61% of obligated updates |
| Claude + typed hint | 0.33 | 1.00 | hint did NOT help |
| Codex naive LLM | 0.34 | 0.99 | misses ~66% |
| Codex + typed hint | 0.33 | 1.00 | hint did NOT help |

## Reading (honest)
- **H1 (strong).** Real deep-research-style agents, given an evidence-world delta, recover only
  ~1/3 of the claims that must change (ACR 0.33–0.39) while leaving unaffected claims intact
  (UCP ~1.0). They behave like a conservative near-flat-ledger: they fix the directly-cited claim
  but miss derived / temporal / recomputed downstream claims. This is the stale-residual failure on
  evidence updates (cf. Mr.DRE), reproduced on two independent model families.
- **Method value (large).** The deterministic typed-graph impact analysis + constrained patch hits
  ACR=UCP=1.0 on the controlled gold, vs full-regeneration (UCP=0.0, the spurious-revision failure)
  and the flat ledger (ACR=0.13). The joint objective cleanly separates the methods.
- **H3 negative, and it sharpens the story.** Simply HANDING the LLM the typed-graph affected set
  did not raise its ACR (0.33 vs 0.39). So the bottleneck is NOT identifying affected claims — even
  told which to change, the agent under-edits (notably it does not reliably recompute derived
  numerics). The contribution is therefore the *structured, mostly-deterministic maintenance
  pipeline* (analyze → constrained patch → verify), not "prompt an agent better."

## Scope / next
Pilot-scale, controlled worlds, deterministic gold, 2 model families. Next: error decomposition
(which claim types are missed), more seeds for tighter CIs, the real-versioned-evidence layer
(gated), and human adjudication (gated). The headline figure is the ACR–UCP plane with methods on
the frontier.
