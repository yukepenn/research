# Four-Paper Agent Research Starter

This package initializes a rigorous four-paper research program:

1. ToolMorph
2. CrossCheck
3. DeltaResearch
4. HarnessGuard

## Start

1. Give the whole folder to Claude Code/Codex.
2. Tell the agent to read `CLAUDE.md` and `RESEARCH_MANUAL.md` in full.
3. Configure authorized model providers, budget limits, and local sandboxing.
4. Run only reversible/local tasks until the human gates in the manual are reached.

The manual is the authoritative research protocol. The templates are scaffolding, not evidence.
